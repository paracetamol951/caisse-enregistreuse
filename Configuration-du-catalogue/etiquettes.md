#? Impression d'étiquettes

Le logiciel vous permet d'imprimer des étiquettes pour vos articles, vos clients et vos tables.

## Contenu des étiquettes

Les étiquettes peuvent contenir :

- Le nom de l'article, du client ou de la table
- Le code-barres ou le QR code
- Le prix de l'article, si disponible
- La quantité, si disponible
- Une description
- La date limite de consommation (DLC), si disponible

## Utilisation des étiquettes

- Scanner le code-barres d'un article ajoute l'article à la commande
- Scanner le code-barres d'un client affecte le client à la commande en cours
- Scanner le code-barres d'une table dans l'interface de vente permet d'associer la table à la commande
- Scanner le QR code d'une table avec un smartphone ouvre la page de commande dédiée à cette table

## Supports d'étiquetage

- Les étiquettes clients peuvent être collées sur des cartes plastiques
- Les étiquettes produits peuvent être collées directement sur les produits ou imprimées sur des feuilles A4 pour être scannées facilement

## Personnalisation de l'impression

Depuis la page d'impression des étiquettes, vous pouvez :

- Utiliser un filtre de recherche pour n'imprimer que certaines étiquettes
- Sélectionner une catégorie d'articles ou de clients
- Définir les marges d'impression, le nombre de colonnes, et l'espacement

Cela vous permet d'ajuster l'impression à votre format d'étiquettes habituel.

? Pensez à tester votre mise en page sur une feuille normale avant d'imprimer sur des étiquettes adhésives.
