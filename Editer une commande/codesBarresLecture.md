# Lecture des codes barres

Lorsque la configuration des codes barres est terminée, vous pouvez commencer à procéder à la saisie des commandes à l\'aide du lecteur de codes barres.

Naviguez simplement dans la page de saisie de commande (clavier), puis commencez à scanner des articles.

## Manipulation pratique avec un scanner laser

Le scanner laser est un achat indispensable pour votre établissement.

- Fluidifie la lecture des codes barres.
- Compatible tous supports.

## Manipulation pratique avec un objectif (camera de smartphone)

Au début il peut paraître difficile d\'effectuer rapidement la lecture des codes barres, mais cela devient vite plus efficace avec un peu d\'entraînement.

- Orienter le code barre horizontalement.
- Maintenir une distance d\'au moins 10 cm du capteur.
- Utiliser un bon éclairage.
- Sur Android, utilisez les boutons volume pour démarrer/arrêter la lecture.
- Assurez-vous que l\'objet est immobile pour une mise au point correcte.

Sur le clavier de caisse, utilisez le bouton \"Code Barre\" pour activer le pointeur de visée.

Lorsqu\'un code est reconnu, votre appareil émet un son (et une vibration si possible).

## Saisie manuelle d'un code

Si un code barre n'est pas détecté, vous pouvez le saisir manuellement.

- Appuyez sur la barre espace.
- Entrez le code au pavé numérique.
- Un son est émis dès qu\'un article correspondant est trouvé.
