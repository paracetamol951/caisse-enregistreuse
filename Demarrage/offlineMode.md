# Mode hors ligne

Le mode hors ligne vous permet de continuer à utiliser l\'application même sans connexion internet. Les commandes sont ensuite automatiquement synchronisées dès que la connexion est rétablie.

Le mode hors ligne ne peut être utilisé que pendant des courtes périodes de temps (max 24h).

## Conditions d'utilisation {#conditions-dutilisation .mt-20}

Le mode hors ligne est disponible uniquement :

- Sur l'application Android
- Sur un ordinateur en utilisant **Chrome** ou **Firefox**

## Fonctionnement

Si aucune connexion internet n'est détectée, le logiciel bascule automatiquement en mode hors ligne.

Dès qu\'une connexion est disponible, le logiciel repasse automatiquement en mode en ligne.

## Fonctionnalités disponibles hors ligne {#fonctionnalités-disponibles-hors-ligne .mt-20}

- Consulter le catalogue d'articles
- Créer des devis ou préparer des commandes
- Consulter votre carnet de clients

?? L\'envoi des factures et l\'enregistrement définitif des commandes ne sont possibles qu\'une fois la connexion internet rétablie.

## Synchronisation après reconnexion {#synchronisation-après-reconnexion .mt-20}

Une fois connecté à internet, une fenêtre vous propose automatiquement de synchroniser les commandes créées hors ligne.

Chaque commande recevra alors un numéro unique et sera enregistrée dans la base de données.
