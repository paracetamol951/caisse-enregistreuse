# Packs d'articles

? Les packs vous permettent de commercialiser des ensembles d'articles groupés.

Il peut s'agir par exemple d'un panier contenant plusieurs produits, ou d'une offre de type : "1 acheté, le second à -50%".

##? Activation de la fonctionnalité

Cette option s'active en page **Configuration \> Options générales \> Options du catalogue d'articles**.

Une fois activée, une nouvelle entrée **Packs** apparaîtra dans le menu Configuration.

? Une licence premium étendue est nécessaire pour utiliser cette fonctionnalité.

## Fonctionnement des packs

Chaque pack peut contenir une liste d'articles ainsi que des suppléments ou réductions associées.

Les packs sont visibles dans un rayon dédié, comme n'importe quel autre produit.

Ils peuvent également être associés à un code barre.

Lorsque vous ajoutez un pack à une commande, les articles du pack sont ajoutés comme s'ils avaient été saisis un par un.

## Détection automatique des packs

Vous pouvez activer la détection automatique en page **Configuration \> Options générales \> Options du catalogue d'articles**.

Dans la page de configuration des packs, vous pourrez alors choisir pour chaque pack s'il doit être appliqué automatiquement.

? Si activée, cette option permet au logiciel de détecter si les articles d'un pack sont ajoutés séparément, puis de regrouper automatiquement ces articles en appliquant la réduction correspondante.

Cela évite de devoir utiliser un code barre spécifique pour le pack.

Si plusieurs packs peuvent être constitués automatiquement à partir des mêmes articles, l'ordre d'enregistrement des packs est utilisé pour déterminer la priorité.

**? Exemple :**

Si deux shampoings sont ajoutés à la commande, le logiciel peut automatiquement appliquer le pack *1 acheté, le deuxième à -50%*.
