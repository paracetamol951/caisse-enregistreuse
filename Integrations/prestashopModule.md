# Module Prestashop Caisse.enregistreuse.fr

Cette page vous guide pour connecter et paramétrer le module Prestashop avec votre caisse enregistreuse.

[Téléchargez le **module Prestashop caisse enregistreuse**.](/Caisse-Enregistreuse/Prestashop/Module/ns_caisseEnregistreusepro.zip)

## 1. Installation du module

1.  [Téléchargez le **module Prestashop caisse enregistreuse**.](/Caisse-Enregistreuse/Prestashop/Module/ns_caisseEnregistreusepro.zip)
2.  Dans votre backoffice Prestashop, ouvrez **Module Manager**.
3.  Cliquez sur **Installer un module**, puis sélectionnez le fichier du module téléchargé.
4.  Une fois installé, cliquez sur **Configurer**.

<span class="image placeholder screenShot" original-image-src="/Caisse-Enregistreuse/Prestashop/Screenshot_Module_Prestashop%20(1).png" original-image-title="">Clé API</span>

## 2. Connexion au compte caisse

Après l?installation du module sur Prestashop, la première étape consiste à naviguer dans l'onglet 'Connexion compte'.

Dans cette page, vous pouvez soit :

- Vous identifier avec votre compte caisse.enregistreuse.fr existant
- Créer automatiquement un compte caisse.enregistreuse.fr qui sera rattaché à votre module automatiquement

## 3. Synchronisation du catalogue

Une fois votre compte connecté, vous avez la possibilité de synchroniser les données de votre Prestashop vers caisse.enregistreuse.fr en naviguant dans l'onglet 'Synchronisation'.

<span class="image placeholder screenShot" original-image-src="/Caisse-Enregistreuse/Prestashop/Screenshot_Module_Prestashop%20(8).png" original-image-title="">Synchronisation de vos données (articles, attributs, catégories, TVAs, ...)</span>

Le logiciel vous liste les modifications qui seront envoyée avant de les appliquer.

4\. Options de configuration

Dans cette page vous pouvez activer l'enregistrement automatique de vos commandes Prestashop dans le logiciel de caisse.

<span class="image placeholder screenShot" original-image-src="/Caisse-Enregistreuse/Prestashop/Screenshot_Module_Prestashop%20(6).png" original-image-title="">Options de configuration</span>

Vous pouvez également configurer une alerte par email

5\. Historique des enregistrements

L'historique des enregistrements vous permet de vérifier l'état de transfert des différentes requêtes correspondants à vos commandes Prestashop.

<span class="image placeholder screenShot" original-image-src="/Caisse-Enregistreuse/Prestashop/Screenshot_Module_Prestashop%20(7).png" original-image-title="">Options de configuration</span>

Vous pouvez également configurer une alerte par email

6\. Configuration des transporteurs

Si vous le souhaitez, vous pouvez personnaliser quel mode de livraison Caisse.enregistreuse.fr sera associé à tel transporteur dans Prestashop.

<span class="image placeholder screenShot" original-image-src="/Caisse-Enregistreuse/Prestashop/Screenshot_Module_Prestashop%20(5).png" original-image-title="">Options de configuration</span>

Si non configuré, le module utilisera la méthode de livraison par défaut.

7\. Configuration des moyens de paiement

Si vous le souhaitez, vous pouvez personnaliser quel moyen de paiement Caisse.enregistreuse.fr sera associé à tel moyen de paiement dans Prestashop.

<span class="image placeholder screenShot" original-image-src="/Caisse-Enregistreuse/Prestashop/Screenshot_Module_Prestashop%20(4).png" original-image-title="">Options de configuration</span>

Si non configuré, le module ne validera pas les commandes, vous devrez compléter la saisie dans le logiciel de caisse

8\. Lancer le point de vente dans Prestashop

Cet onglet vous permet d'accéder à votre point de vente directement dans Prestashop pour le vente en personne.

<span class="image placeholder screenShot" original-image-src="/Caisse-Enregistreuse/Prestashop/Screenshot_Module_Prestashop%20(3).png" original-image-title="">Options de configuration</span>
