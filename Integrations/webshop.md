# Boutique en ligne

Proposez à vos clients de commander en ligne ou directement depuis votre établissement grâce à votre boutique connectée.

## Modes d'accès à la boutique en ligne

Votre boutique en ligne peut être utilisée de plusieurs manières :

- Sur une tablette ou un écran à l'entrée de votre magasin (bornes de commande)
- Depuis une adresse web dédiée, accessible à vos clients
- Intégrée directement dans votre propre site internet sous forme de panier d\'achat

## Intégration avec le logiciel de caisse

Toutes les ventes passées via la boutique en ligne sont automatiquement intégrées dans le système

- Apparition immédiate dans l'historique des commandes
- Inclusion dans les rapports de vente
- Affichage dans la gestion des préparations de commandes

Lorsqu'une commande est passée, vous pouvez recevoir une alerte par email.

## Personnalisation de l'interface

Vous pouvez modifier les textes et les couleurs de la boutique en ligne :

- Connectez-vous avec un compte administrateur
- Utilisez le bouton de prévisualisation dans la page de configuration
- Cliquez directement sur les textes pour les modifier

## Activer ou désactiver les fonctionnalités

Depuis la page de configuration de la boutique en ligne, vous pouvez activer ou désactiver les différentes options disponibles selon vos besoins.

## Configurer l'adresse de votre boutique

- **Nom de domaine enregistreuse.fr :** Vous pouvez choisir une adresse du type `xxxxx.b.enregistreuse.fr`. La configuration est automatique et immédiate.
- **Nom de domaine personnalisé :** Vous pouvez utiliser votre propre nom de domaine. Pour cela :
  - Modifiez l'entrée DNS `CNAME` de `www.mon-nom-de-domaine.com` et `mon-nom-de-domaine.com`
  - Pointez ces deux entrées vers `caisse.enregistreuse.fr`

La boutique en ligne est un outil puissant pour développer vos ventes, fidéliser vos clients, et fluidifier vos processus de commande.
