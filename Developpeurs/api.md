# API

## Télécharger un export de ventes

Il est possible de récupérer les rapports de ventes d'une journée spécifique via l'API.

Les formats disponibles sont : HTML, PDF, CSV, XML, XLS

L'URL à appeler est indiquée en page Configuration \> Webservices

## Télécharger la liste des articles, rayons, clients, etc

L'API permet aussi d'exporter le catalogue complet au format CSV, HTML ou JSON.

L'URL d'accès est également disponible depuis la page Configuration \> Webservices.
