#? Périphériques

Cette page vous guide dans la configuration des périphériques compatibles : imprimantes, tiroirs-caisses, lecteurs de codes-barres et équipements Sunmi ou Epson.

##? Compatibilité Sunmi (imprimante intégrée, écran secondaire)

Les appareils Sunmi sont automatiquement détectés par l'application.

- L'imprimante thermique intégrée est immédiatement prête à l'emploi
- L'écran client est pris en charge s'il est présent
- Aucune configuration manuelle requise

? Modèles testés : V2, V2s, T2 Mini, P2 Pro

##? Imprimantes Epson (Bluetooth / Ethernet)

Les imprimantes thermiques Epson (modèles ESC/POS) sont compatibles avec l'application.

1.  Allumez l'imprimante Epson
2.  Depuis l'application, ouvrez le menu Paramètres \> Imprimante
3.  Appuyez sur le bouton de recherche pour détecter les périphériques
4.  Choisissez l'imprimante Epson détectée (via Bluetooth ou réseau)

? La détection automatique fonctionne sous Android et iOS.

## Tiroir-caisse StarMicronics (Bluetooth)

Les tiroirs StarMicronics se connectent automatiquement via Bluetooth.

1.  Ouvrez le menu Configuration \> Options générales
2.  Dans la section Matériel, appuyez sur le bouton de détection
3.  Le tiroir Bluetooth apparaît dans la liste. Sélectionnez-le.

? Aucune configuration technique nécessaire -- la détection est automatique.

## Lecteurs de code-barres

- Connectez un lecteur USB ou Bluetooth (reconnu comme clavier)
- Scannez un produit pour l'ajouter au ticket de caisse
- Vous pouvez aussi utiliser l'appareil photo du téléphone

? Aucun pilote requis pour les lecteurs classiques.

## Résumé de compatibilité

  Périphérique                   Connexion           Config
  ------------------------------ ------------------- ----------------------------------------------------
  Sunmi (imprimante, écran)      Interne / auto      Aucune -- détecté automatiquement
  Epson (Bluetooth / Ethernet)   BT / Réseau         Recherche via bouton dans Paramètres \> Imprimante
  StarMicronics (tiroir)         Bluetooth           Détection via Options générales \> Matériel
  Lecteurs de code-barres        USB / BT / Caméra   Plug & Play
