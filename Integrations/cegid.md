# Transférer les ventes vers Cegid Loop

Le logiciel permet d'exporter vos ventes au format FEC, compatible avec Cegid Loop.

## 1.Exporter les ventes au format FEC

Une page d'aide dédiée vous guide pas à pas pour générer un fichier FEC depuis le logiciel. Une fois le fichier FEC exporté, vous pouvez l'utiliser pour l'import dans Cegid.

? Voir : [Rapports de vente](/graphiques.md)

## 2.Importer le fichier dans Cegid Loop

Depuis Cegid Loop, vous avez deux options pour l'importation du fichier : manuelle ou automatique.

### Import manuel

1.  Ouvrez Cegid Loop
2.  Allez dans : Comptabilité \> Imports \> Écritures \> Personnalisées, FEC, TRA
3.  Sélectionnez votre fichier FEC exporté
4.  Validez pour lancer l'import

### Import automatique (en masse)

Vous pouvez également déposer les fichiers dans le répertoire prévu pour l'import automatique, si votre instance de Cegid Loop le permet.

Limites recommandées par Cegid Loop pour les imports automatiques :

- Jusqu'à 1 million de mouvements
- Fichier jusqu'à 133 Mo

?? Les temps de traitement peuvent être longs pour les fichiers volumineux (plusieurs heures).

## Conseils pratiques

- Vérifiez que votre export FEC est bien structuré avant l'import.
- Conservez toujours une copie de vos fichiers exportés pour archivage.
- Utilisez l'import manuel si vous avez un doute sur la compatibilité.
