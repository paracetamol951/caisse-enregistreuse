#? Marque blanche

Le logiciel propose une fonctionnalité de marque blanche destinée aux revendeurs souhaitant proposer la caisse enregistreuse à leurs clients sous leur propre identité.

## Principe de fonctionnement

Vous facturez directement vos clients pour l\'utilisation de la caisse, et vous vous chargez de régler les licences auprès de nous.

Vous créez un compte boutique pour votre client, en effectuez la configuration, puis activez l\'option « Marque blanche ».

## Accès client personnalisé

Une URL d'accès est définie pour chaque client.

Votre client utilisera uniquement cette adresse pour accéder à sa caisse.

Il ne pourra pas se connecter à d'autres comptes.

Si l'accès se fait via une application, configurez les applications et activez le verrouillage dans la configuration Marque blanche.

## Gestion des demandes d'assistance

Lorsque le client utilise le formulaire de contact, les messages vous sont envoyés directement.

Vous pouvez ensuite nous transmettre les demandes si nécessaire.

## Configuration du domaine

Deux possibilités s'offrent à vous :

- Utiliser votre propre nom de domaine : créer une entrée CNAME vers enregistreuse.fr sur votre domaine et sur www.votredomaine.ext
- Utiliser un sous-domaine \*.p.enregistreuse.fr (par exemple perfeito.p.enregistreuse.fr), qui est configuré automatiquement et gratuit.

Dans les deux cas, vous devez indiquer l'adresse choisie dans la page de configuration de la boutique, onglet « Marque blanche ».

## Facturation client

Vous êtes en charge de la facturation et du support de vos clients.

Vous pouvez fixer librement vos tarifs, ajouter des prestations (installation, formation, support...), et choisir vos moyens de paiement.

Vous devez disposer d'un statut professionnel (auto-entrepreneur, société...).

## Gestion des comptes

- Créer le compte boutique pour le client
- Indiquer qu'il s'agit d'un compte en marque blanche
- Créer un ou plusieurs comptes utilisateur (au moins un admin pour le client)
- Associer votre compte utilisateur à ce compte boutique

Depuis votre compte, vous pourrez régler les licences, modifier les paramètres de marque blanche, et gérer l'ensemble de vos clients.

## Faire un test

Vous pouvez créer un nouveau compte test, activer la marque blanche, et vérifier le bon fonctionnement.

Dans les paramètres, sélectionnez l'utilisateur qui aura les droits sur la marque blanche (vous).
