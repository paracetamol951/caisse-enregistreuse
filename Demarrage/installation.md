#? Démarrage

Cette section vous guide dans les premières étapes : installation de l'application, création de votre établissement, et découverte de l'interface de vente.

## 1. Installation de l'application Android

1.  Ouvrez le Google Play Store
2.  [Recherchez : caisse.enregistreuse.fr](https://play.google.com/store/apps/details?id=com.caisse.enregistreuse2&downloadInvoice=1){target="_blank"}
3.  Touchez Installer, puis ouvrez l'application

**Note :** la langue de l'application est automatiquement détectée selon la langue de votre appareil Android.

## 2. Installation de l'application iOS

1.  Ouvrez l'App Store sur votre iPhone ou iPad
2.  [Recherchez : caisse.enregistreuse.fr](https://apps.apple.com/fr/app/cash-register-pos/id1259175837?downloadInvoice=1){target="_blank"}
3.  Installez l'application, puis ouvrez-la

? L'application iOS offre une expérience similaire à la version Android.

##? 3. Installation depuis le Microsoft Store

1.  Ouvrez le Microsoft Store sur votre ordinateur Windows
2.  [Recherchez : caisse.enregistreuse.fr](https://apps.microsoft.com/detail/9nblgggztq9k?hl=fr-FR&gl=FR&downloadInvoice=1){target="_blank"}
3.  Cliquez sur Installer, puis lancez l'application depuis le menu Démarrer

? La version Windows est idéale pour une utilisation avec un écran tactile ou un terminal de point de vente.

## 4. Utilisation dans un navigateur Web

1.  Ouvrez votre navigateur (Chrome, Edge, Firefox...)
2.  Allez à l'adresse : [https://caisse.enregistreuse.fr](https://caisse.enregistreuse.fr){target="_blank"}
3.  Connectez-vous avec votre adresse email et mot de passe

? Vous pouvez accéder à 99% des fonctionnalités depuis un navigateur, sans installer d'application.

## 5. Création du commerce (compte)

Au premier lancement, vous devez créer un compte :

1.  Saisissez un email valide
2.  Indiquez le nom de votre établissement (ex. : La petite épicerie)
3.  Validez : un mot de passe vous est envoyé par email

? Votre commerce est enregistré sur le cloud et lié à votre compte. Vous pouvez vous reconnecter sur n'importe quel appareil.

?? **Le mode test est activé par défaut**. Il vous permet de tester la configuration sans enregistrer de ventes réelles.

##? 6. Présentation de l'interface de vente

- Élément                      Description
  ------------------------------- ----------------------------------------------------------------------------------------------------------------------------
- Bloc gauche                  Affiche au choix : rayons, groupes de rayons, articles, clients, tables ou réductions. L\'utilisateur choisit l'affichage.
- Ticket (colonne de droite)   Contient les articles de la commande en cours
- Bouton "Ajouter"             Démarre une nouvelle commande
- Encaisser                    Finalise la commande avec un paiement
- Menu                          Paramètres, stock, rapports, connexion cloud, etc.

## Astuce

Vous pouvez tester l'application sans risque grâce au mode test. Une fois votre configuration prête, vous pouvez le désactiver dans les paramètres pour commencer à enregistrer des ventes réelles.
