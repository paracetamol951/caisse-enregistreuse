# Parrainage client

Le système de parrainage permet à vos clients de recommander votre boutique à leurs proches tout en recevant des récompenses.

## Activation du parrainage

Pour activer le parrainage client, allez dans :

**Configuration \> Options générales \> Fidélisation**

## Montants offerts

Vous pouvez configurer :

- Le montant offert au parrain
- Le montant offert au filleul

Ces montants sont attribués sous forme d'avoir utilisable en caisse.

## Fonctionnement du code de parrainage

- Chaque client dispose d'un code de parrainage unique et permanent.
- Ce code est imprimé automatiquement sur les tickets de caisse.
- Il peut aussi être inclus dans les emails ou SMS de confirmation de commande.
- Le code peut être saisi lors de la création d'un compte client, sur la boutique en ligne ou dans la caisse.

? Encouragez vos clients à partager leur code pour bénéficier du parrainage !
