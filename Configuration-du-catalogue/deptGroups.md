# Groupes de rayons

? Si vous disposez d'un trop grand nombre de rayons, vous pouvez activer l'option **Groupes de rayons** dans la page de configuration générale.

? Cette option se trouve dans **CONFIGURATION \> Options générales \> Options du catalogue d'articles**.

##? Fonctionnement

Une fois l'option activée, vous pouvez créer une liste de groupes de rayons, puis classer chaque rayon dans l'un de ces groupes.

Cette classification permet une navigation plus fluide dans l'interface de vente.

? L'interface de la caisse enregistreuse proposera alors de sélectionner un groupe de rayons avant d'afficher les rayons associés.

? Cette fonctionnalité est particulièrement utile pour les commerces ayant un large catalogue d'articles répartis dans de nombreux rayons.
