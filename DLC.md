# Dates limites de consommation (DLC)

? Le logiciel vous permet de gérer les dates limites de consommation de vos articles (DLC).

##? Activation de la fonctionnalité

Pour activer cette fonctionnalité, rendez-vous dans la page **Configuration \> Options générales**, puis activez l'option dans le paragraphe **Options du catalogue d'articles**.

Vous pouvez également définir un **délai d'avertissement** (en jours) avant d'atteindre la date limite de consommation d'un article.

## Configuration des DLC pour les articles

Une fois l'option activée, vous pourrez définir la date limite de consommation pour chaque article depuis la page de configuration des articles.

? Si vous vendez un même article avec plusieurs dates limites de consommation différentes, vous devez leur attribuer des codes-barres distincts afin de les différencier.

## Suivi des articles proches ou dépassés

Dans la page de fermeture de caisse, le logiciel vous affiche automatiquement :

- Les articles approchant de leur date limite (signalés dans la couleur d'avertissement)
- Les articles dont la DLC est dépassée (affichés en rouge)
