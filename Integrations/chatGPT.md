# Connectez ChatGPT avec caisse.enregistreuse.fr

Ce module vous permet de connecter ChatGPT à votre logiciel de caisse, et de donner accès à ChatGPT à vos données de ventes, ainsi que toutes les autres données, et la capacité d'enregistrer des ventes dans le logiciel

Ce module est actuellement en version beta. Compatible uniquement sur PC ou Mac.

Ce module nécessite un abonnement ChatGPT 'Espace de travail' pour pouvoir fonctionner

## Installation

Dans ChatGPT, en utilisant un Espace de travail, accédez aux paramètres, Connecteurs, puis ajoutez un nouveau connecteur avec les paramètres suivants

<span class="image placeholder screenShot" original-image-src="/Caisse-Enregistreuse/Help/ChatGPT_NewConnector.png" original-image-title=""></span>

| Variable       | Value                             |
|----------------|-----------------------------------|
| Name           | Caisse enregistreuse              |
| Description    | Can record sales and view reports |
| MCP Server URL | https://mcp.kash.click/mcp        |
| Authentication | oAuth                             |
