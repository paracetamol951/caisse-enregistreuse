# Éditer une note de frais

Dans votre logiciel de caisse, vous pouvez facilement générer des notes de frais pour vos clients en quelques étapes simples. Voici comment procéder :

## Ajout du bouton de note de frais {#ajout-du-bouton-de-note-de-frais .mt-4}

Avant de pouvoir éditer des notes de frais, vous devez ajouter un bouton dédié dans l'interface. Rendez-vous dans la page de configuration, section 'Actions'. Là, ajoutez un nouveau bouton d'action en choisissant le type 'Note de frais'. Une fois ce bouton ajouté, il apparaîtra dans l'interface de vente principale.

## Utilisation du bouton de note de frais {#utilisation-du-bouton-de-note-de-frais .mt-4}

Lorsque vous avez une commande validée en cours, vous pouvez simplement cliquer sur le bouton 'Note de frais'. Cela générera automatiquement une note de frais pour le montant total de la commande.

Si vous souhaitez éditer une note de frais uniquement pour certains articles, cliquez directement sur les articles concernés dans le ticket de caisse, puis appuyez sur le bouton 'Note de frais'. Vous obtiendrez ainsi une note de frais pour le montant total des articles sélectionnés, avec la TVA correspondante.

## Éditer une note de frais par paiement ou montant manuel {#éditer-une-note-de-frais-par-paiement-ou-montant-manuel .mt-4}

Une autre option consiste à cliquer sur une ligne de paiement déjà encaissé dans le ticket de caisse. Après avoir sélectionné les paiements souhaités, appuyez sur 'Note de frais'. La note de frais sera alors établie pour le montant total des paiements sélectionnés, avec une répartition de la TVA au prorata.

Enfin, vous pouvez également saisir manuellement un montant pour la note de frais (qui doit être inférieur ou égal au montant total de la commande). Une fois le montant saisi, appuyez sur le bouton 'Note de frais' pour obtenir la note correspondante, avec une répartition de la TVA au prorata également.

Voilà, vous savez maintenant comment éditer facilement des notes de frais pour vos clients dans différentes situations !
