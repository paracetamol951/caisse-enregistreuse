# Effectuer un remboursement

Le logiciel vous permet de gérer différents types de remboursements, selon que la commande soit encore en cours, partiellement ou totalement payée.

?? Vous devez disposer des droits administrateur ou manager pour pouvoir effectuer un remboursement.

## Remboursement simple

Pour effectuer un remboursement rapide d'un montant connu :

- Saisissez le montant à rembourser sur le pavé numérique.
- Sélectionnez le mode de paiement à utiliser pour le remboursement.

Le montant remboursé s'affiche en rouge sur le ticket de caisse. Une vente de valeur négative est automatiquement enregistrée pour la journée, avec la mention \"remboursement\".

## Remboursement d'articles enregistrés

Pour rembourser des articles spécifiques :

- Sélectionnez le rayon concerné à l'aide de l'encart des rayons.
- Cliquez sur les articles à rembourser.
- Choisissez un mode de remboursement pour valider l\'opération.

## Remboursement d'une commande complète

Si vous souhaitez rembourser une commande dans son intégralité :

- Rendez-vous dans l'historique et chargez la commande concernée.
- Sélectionnez le mode de remboursement à l'aide des boutons du clavier.

Le logiciel crée alors automatiquement une nouvelle commande négative avec les mêmes articles que la commande initiale.

## Remboursement partiel d'une commande

Si seulement certains articles doivent être remboursés :

- Chargez la commande depuis l'historique.
- Sélectionnez les articles à rembourser sur le ticket de caisse.
- Sélectionnez un mode de remboursement.

Le logiciel génère une commande négative à la date du jour, contenant uniquement les articles sélectionnés.

? Astuce : tous les remboursements apparaissent dans vos rapports comme des ventes négatives, ce qui facilite le suivi comptable.
