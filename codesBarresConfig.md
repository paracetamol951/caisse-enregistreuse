# Configuration des codes barres

?? Afin d'effectuer cette configuration, vous devez vous rendre dans la page **CONFIG \> ARTICLES**.

## Ajouter un nouvel article à l'aide de son code barre

Scannez simplement un nouveau code barre depuis la page de configuration des articles.

Une fenêtre s'ouvrira automatiquement pour vous inviter à saisir les informations de l'article : prix, nom, rayon, etc.

? Cette méthode permet de créer rapidement une fiche article à partir d'un code barre.

## Attacher un code barre à un article existant

Dans le tableau des articles, cliquez sur la cellule de la colonne **Code Barre** correspondant à l'article souhaité.

Il vous sera alors demandé de scanner le nouveau code barre à associer à cet article.

## Rechercher un article à l'aide de son code barre

Pour retrouver rapidement un article dans la configuration, scannez son code barre directement depuis la page de configuration.

L'article correspondant s'affichera immédiatement.
