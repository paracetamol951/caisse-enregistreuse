# Configurer le ticket de caisse

## Personnalisation de l'en-tête et du pied de page

Dans la page de configuration générale, vous pouvez définir un logo en en-tête ainsi qu'un texte personnalisé en haut et en bas du ticket.

- Formats acceptés : PNG, JPG, GIF
- L'image est redimensionnée automatiquement

## Personnalisation des informations affichées sur le ticket

Dans la page de configuration générale, un très grand nombre d\'options sont disponibles afin de personnaliser les informations qui seront affichées sur le ticket de caisse.
