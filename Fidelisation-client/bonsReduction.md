# Bons de réduction

Les bons de réduction permettent à vos clients de bénéficier d'une remise, sous conditions. Ils peuvent être utilisés sur place ou en ligne.

## Activer les bons de réduction

Pour utiliser cette fonctionnalité, vous devez d'abord activer l'option correspondante en page Config, Options générales, onglet Fidélisation.

## Configurer un bon de réduction

Une fois la fonctionnalité activée, vous pouvez créer vos bons de réduction depuis la page Configuration \> Bons de réduction.

Chaque bon de réduction est identifié par un code unique que le client pourra saisir pour en bénéficier.

## Conditions d'application d'un bon

Vous pouvez définir des conditions pour que le bon soit accepté. Parmi les options disponibles :

- Nombre maximum d'utilisations totales
- Montant minimum de commande
- Commande contenant un article spécifique
- Commande contenant un article d'un rayon spécifique
- Date de début et/ou de fin de validité
- Utilisable uniquement par un client ou groupe de clients
- Usage unique par client
- Nombre minimum d'articles dans la commande
- Restriction à une caisse spécifique
- Restriction à une méthode de livraison

## Choisir la réduction appliquée

Vous devez sélectionner une réduction à appliquer lorsque le bon est valide. Seules les réductions en pourcentage ou montant fixe sont autorisées (hors réductions de type menu ou prix fixé).

La réduction est appliquée sur le sous-total de la commande.

## Utilisation du bon

Les clients peuvent saisir leur code en ligne ou via le bouton d'action 'Bon de réduction' dans l'interface de vente.

Le logiciel vérifie automatiquement si toutes les conditions sont réunies avant d'appliquer la réduction.

Un même bon ne peut être utilisé qu'une seule fois par commande.

## Envoyer un bon de réduction

Depuis la page de configuration des bons de réduction, un bouton 'Envoyer' permet de lancer une campagne marketing (email, SMS...) pour communiquer le code du bon aux clients ciblés.

Les bons de réduction sont un outil puissant pour fidéliser vos clients et dynamiser vos ventes pendant les périodes stratégiques.
