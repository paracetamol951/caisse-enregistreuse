# Paiement partiel

Lorsque votre commande est constituée, méthode pour enregistrer un paiement partiel, ou paiement en plusieurs fois

Saisissez le montant payé (au clavier ou à l\'aide de pavé numérique de l\'interface), puis sélectionnez le mode de paiement.

Le logiciel vous affichera automatiquement le montant restant à payer, vous pouvez ensuite recommencer l\'opération pour enregistrer d\'autres paiements.
