# Multiples lieux de préparation

Le logiciel permet de configurer plusieurs lieux de préparation pour organiser la production de vos commandes.

## Principe

Chaque lieu de préparation peut être associé à un type d'article spécifique.

Cela permet de répartir les tâches entre plusieurs espaces physiques selon le type de produit vendu.

## Exemple d'usage

- Les articles de type nourriture sont préparés dans une cuisine.
- Les articles de type boisson sont préparés au bar.

Chaque zone voit uniquement les articles qui lui sont affectés.

?? Cette configuration se fait depuis la page de configuration des articles, où vous pouvez associer chaque article à un lieu de préparation.
