# Configurer les rayons

? Pour effectuer cette opération, vous devez disposer des droits administrateur.

?? Accédez à la page **Config** , puis au sous-menu **Rayons**.

## Ajouter un nouveau rayon

Cliquez sur le bouton « Ajouter » situé en haut du tableau.

Remplissez les champs proposés dans la fenêtre qui s'affiche.

## Modifier un rayon existant

- Double-cliquez sur le champ à modifier.
- Appuyez sur Entrée pour valider votre modification.
- Les modifications sont enregistrées automatiquement après chaque saisie.
- Pour modifier l'image du rayon, cliquez sur l'image existante et remplissez le formulaire affiché.

## Signification des champs

- **Nom** : Nom du rayon tel qu'il apparaîtra sur le ticket de caisse et sur le clavier.
- **Taux de TVA** : Taux de TVA appliqué par défaut aux articles de ce rayon.
- **Réduction** : Réduction appliquée par défaut aux articles de ce rayon.
- **Clavier** : Indique si une touche de sélection est affichée dans le clavier de vente pour ce rayon. Sinon, il devra être sélectionné via une liste déroulante.
- **Image** : Image affichée sur la touche du clavier (format JPG ou PNG, taille \< 2 Mo).
