# Gestion des calories

?? Le logiciel vous permet d'enregistrer le nombre de calories contenues dans chaque article.

? Le total des calories de la commande ainsi que les calories par article peuvent être affichés sur le ticket de caisse.

## Activation de la fonctionnalité

Pour activer cette option, rendez-vous dans la page **Configuration \> Options générales \> Commandes**, puis cochez l'option *« Activer la gestion des calories »*.

## Configuration des calories des articles

Une fois la fonctionnalité activée, vous pourrez saisir le nombre de calories pour chaque article en vous rendant dans la page de configuration des articles.

?? Cette information est utile pour les établissements soumis à une obligation d'affichage nutritionnel, ou pour sensibiliser la clientèle.
